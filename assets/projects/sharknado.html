Sharknado Animation

This animation written in C++ originally started off as just a model of a house that the user could rotate objects and change into different perspectives. With a growing understanding of the OpenGL Utility Toolkit (GLUT), I was able to add many more features and functionality eventually turning into this animation which was inspired by the ridiculous Sharknado movie.
